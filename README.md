# LEGO Database Hub

LEGO Database Hub is a project developed and created to organize, showcase and keep track of my personal collection of LEGO sets, cataloged in strict chronological order of purchase.

Every new set that arrives, once it has been fully assembled, immediately finds its place in this gallery, which grows alongside me and my passion.

It is not and never will be a truly complete project: just like the collection itself, it is constantly evolving, enriched with new bricks, new minifigures, and new details.🧱

## 📋 Description

**LEGO Database Hub** is a **single‑page application** (SPA) built with pure HTML, CSS, and JavaScript.  
All data – name, set number, piece count, theme, and image – are stored in a JavaScript array and displayed in a dynamic grid of cards.

The application includes features to enhance usability:

- **Language switch** (Italian ↔ English) with instant translation of names and the "pieces" label.
- **Dark/Light theme** with an iOS‑style toggle showing ☀️ and 🌙 emojis.
- **Set images** dynamically loaded via **jsDelivr** from a GitHub repository, with fallback to thematic emojis.
- **Mobile‑first design** – the grid adapts to every screen, from smartphone to desktop.
- **Custom favicon** served via jsDelivr.

## ✨ Key Features

| Feature                | Description                                                                                     |
|------------------------|-------------------------------------------------------------------------------------------------|
| **Grid gallery**       | Each set is displayed as a card with image, name, set number, piece count, and theme.           |
| **Bilingual (IT/EN)**  | Switch languages with one click: names and the label "pieces" update instantly.                 |
| **Dark/Light mode**    | Elegant toggle that adapts background, text, card, and detail colors to the selected theme.     |
| **CDN images**         | Images are loaded from jsDelivr using the GitHub repository, with lazy loading.                 |
| **Favicon**            | Custom browser tab icon served via CDN.                                                         |
| **Responsive**         | Layout optimized for desktop, tablet, and smartphone (dedicated breakpoints).                   |

## 🛠️ Technologies Used

- **HTML5** – semantic structure.
- **CSS3** – CSS variables for theming, **CSS Grid** layout, smooth transitions.
- **JavaScript (Vanilla)** – state management, dynamic rendering, event listeners.
- **jsDelivr** – CDN for serving favicon and images from the GitHub repository.
- **GitHub** – hosting the repository and static assets.
